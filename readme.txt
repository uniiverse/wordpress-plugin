Universe.com integration for WordPress.

== Description ==

The Universe.com plugin for WordPress! Easily add links to sell tickets
directly on your WordPress-driven event site. Using this plugin, transform your
links to universe.com events into your own, on-site boxoffice!

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the contents to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Include a link to a universe.com event URL somewhere on your site
4. The link will now open a custom buy tickets modal on your site!
